Project Overview
This project focuses on analyzing bank loan data to understand loan application trends, funding performance, repayment behavior, and overall loan portfolio quality. 
An interactive Power BI dashboard is developed to support data-driven decision-making for banking stakeholders.

Objectives
Analyze total loan applications and monthly trends (MTD & MoM)
Evaluate total funded amount and repayment performance
Assess loan quality using Good vs Bad loan classification
Study borrower behavior across grades, home ownership, and regions
Identify risk patterns and portfolio distribution

Key KPIs
Total Loan Applications
Month-to-Date (MTD) Loan Applications
Month-over-Month (MoM) Growth in Applications
Total Funded Amount (MTD & MoM)
Total Amount Received (MTD & MoM)
Good Loan % and Bad Loan %

Dimensions Analyzed
Loan Purpose
Home Ownership
Loan Status (Good vs Bad)
Time (Month).
